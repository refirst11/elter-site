# cssx.set
In set a single syntax is supported. There's no need to think about classNames here.  
we recommend using the variable name css + subject
## Write set and compile 
The hash of the object is directly attached to the className.

<Tabs items={['tsx', 'css' ]}>
  <Tab>
    ```tsx title="Something.tsx"
    import cssx from 'typedcssx'

    const cssLink = cssx.set({
      fontSize: '14px',
      color: 'var(--color-link)',
      textDecoration: 'none'
    })

    const cssBox = cssx.set({
      display: 'block',
      padding: 20,
      width: 60,
      height: 60
    })

    const cssContainer = cssx.set({
      display: 'flex',
      justifyContent: 'center',
      alignItems: 'center'
    })
    ```
  </Tab>
