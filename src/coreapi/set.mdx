# Style.set
In set you define the properties directly, there's no need to worry about classNames here, we recommend using the variable name ~Style.
```ts title="style.css.ts"
import Style from 'typedcssx'

export const linkStyle = Style.set({
  fontSize: '14px',
  color: 'var(--color-link)',
  textDecoration: 'none'
})

export const boxStyle = Style.set({
  display: 'block',
  padding: 20,
  width: 60,
  height: 60
})

export const containerStyle = Style.set({
  display: 'flex',
  justifyContent: 'center',
  alignItems: 'center'
})
```
