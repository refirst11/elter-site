## Build setting

Add the compile command to the `package.json` script field.

<Tabs>
  {/* prettier-ignore */}
  <Tab>
  ```json title="package.json"
    "scripts": {
      "build": "npm run compile && next build",
      "compile": "cd node_modules/typedcssx && npm run compile"
    },
  ```
  </Tab>
</Tabs>

### tsconfig.json
Add **/*css.ts and **/*css.tsx to icludes in tsconfig.json
<Tabs>
  <Tab>
  ```json title="tsconfig.json"
    "include": ["**/*css.ts", "**/*css.tsx"],
  ```
  </Tab>
</Tabs>

## Global style sheet
Load global.css in layout.tsx, which is the root directory of the app.
<Tabs>
  <Tab>
```tsx title="layout.tsx"
import '../../node_modules/typedcssx/dist/core/styles/global.css'
```
  </Tab>
</Tabs>

## Support Node.js v18 - 22

This project has been tested and is confirmed to work with Node.js versions from v18 to v22. Please you are using one of these versions for optimal compatibility and performance.
