# Style class

## Everything starts with this

### Style create
Create a stylesheet that takes the key as a className.  
Since this function is created almost only once, we recommend using styles as the variable name.
```ts title="style.css.ts"
import { Style } from 'typedcssx'

export const styles = Style.create({
  container: {
    display: 'flex',
    justifyContent: 'space-between'
  },
  link: {
    fontSize: '14px',
    textDecoration: 'none'
  },
})
```
### Compiled create
A hashed prefix of the object is added to the class name, so styles do not conflicts between different components.
```css title="css"
.container_40ExT {
  display: flex;
  justify-content: space-between;
}

.link_40ExT {
  font-size: 14px;
  text-decoration: none;
}
```

### Style global
Global css can be defined here.  
There is no need to put it in a variable, you can preview it by importing it to 'use client' and loading it.
```ts title="global.css.ts"
Style.global({
  body: {
    color: 'var(--color-text)',
    background: 'var(--color-bg)',
    margin: '0 auto'
  },
  h2: {
    color: 'var(--color-heading)',
    fontSize: '30px',
    fontWeight: '550'
  },
  ".headings": {
    color: 'var(--color-heading)',
    fontSize: '24px',
    fontWeight: '450'
  }
});
  ```

### Compiled global
In addition to html elements, global allows you to directly define attributes and class names for pseudo elements and class.
```css title="css"
body {
  color: var(--color-text);
  background: var(--color-bg);
  margin: 0 auto;
}

h2 {
  color: var(--color-heading);
  font-size: 30px;
  font-weight: 550;
}

.headings {
  color: var(--color-heading);
  font-size: 24px;
  font-weight: 450;
}
```

### Style set
In set you define the properties directly, there's no need to worry about classNames here, we recommend using the variable name ~Style.
```ts title="style.css.ts"
import { Style } from 'typedcssx'

export const linkStyle = Style.set({
  fontSize: '14px',
  color: 'var(--color-link)',
  textDecoration: 'none'
})

export const boxStyle = Style.set({
  display: 'block',
  padding: 20,
  width: 60,
  height: 60
})

export const containerStyle = Style.set({
  display: 'flex',
  justifyContent: 'center',
  alignItems: 'center'
})
```

### Compiled set
The hash of the object is directly attached to the className.
```css title="css"
._jo139 {
  font-size: 14px;
  color: var(--color-link);
  text-decoration: none;
}

._AAbLv {
  display: block;
  padding: 20px;
  width: 60px;
  height: 60px;
}

._Ux5Y4 {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

### Style root
The root was created primarily to manage CSS variables.
```ts title="global.css.ts"
Style.root({
  "&[data-theme='dark']": {
    '--color-text': 'rgb(180 180 190)',
    '--color-bg': 'rgb(42 43 43)',
    '--color-heading': 'rgb(230, 255, 247)',
    '--color-border': 'rgba(30, 30, 30, 0.43)',
    '--color-link': 'rgb(100 200 200)',
    '--color-list': 'rgba(80, 87, 80, 0.3)',
    '--color-sub': 'rgb(220 220 220)',
    '--color-card': 'rgba(33, 33, 33, 0.3)',
    '--color-logo': 'rgb(100 220 200)',
    '--color-accent': 'rgb(206 206 206)',
  },

  "&[data-theme='light']": {
    '--color-text': '#4b5563',
    '--color-bg': 'rgb(255, 255, 255)',
    '--color-heading': 'rgb(70, 86, 86)',
    '--color-border': '#eaeaea',
    '--color-link': 'rgb(60, 135, 232)',
    '--color-list': '#e0efff',
    '--color-sub': '#222',
    '--color-card': 'rgb(245 245 245)',
    '--color-logo': 'rgb(60 165 140) ',
    '--color-accent': 'rgb(70, 86, 86)',
  }
})
```
