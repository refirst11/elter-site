# keyframes

@keyframes can be defined in cssx.global.
<Tabs items={[ 'global.ts', 'css']}>
  <Tab>
```ts title="global.ts"
cssx.global({
  '@keyframes fade': {
    from: {
      opacity: 0,
    },
    to: {
      opacity: 1,
    },
  },
  '@keyframes move': {
    '0%': {
      transform: 'translate(0px)'
    },
    '100%': {
      transform: 'translate(100px)'
    },
  },
});
```
  </Tab>
  <Tab>
```css title="css"
@keyframes fade {
  from {
    opacity: 0;
  }
  to {
    opacity: 1;
  }
}

@keyframes move {
  0% {
    transform: translate(0px);
  }
  100% {
    transform: translate(100px);
  }
}
```  
  </Tab>
</Tabs>



## Uses
The created animation can be used with cssx.create, cssx.set, etc.
<Tabs items={[ 'tsx', 'css']}>
  <Tab>
```ts title="Something.tsx"
const css = cssx.create({
  box: {
    padding: '20px',
    animation: 'move 2s'
  },
  heading: {
    fontSize: 24,
    animation: 'fade 3s infinite'
  }
});
```
  </Tab>
  <Tab>
```css title="css"
.box_ocinp0 {
  padding: 20px;
  animation: move 2s;
}

.heading_ocinp0 {
  font-size: 24px;
  animation: fade 3s infinite;
}
```  
  </Tab>
</Tabs>
